🚀 3 步部署

Step 1：上傳到 GitHub
  1. 訪問 https://github.com/new
  2. 輸入名稱：law-format-converter
  3. 建立 repo
  4. 用「Add file」→「Upload files」上傳 3 個檔案

Step 2：連接 Cloudflare Pages
  1. 訪問 https://pages.cloudflare.com
  2. 點「Create a project」
  3. 選「Connect to Git」
  4. 選擇 law-format-converter repo
  5. 點「Deploy」

Step 3：等待完成
  1-2 分鐘內，你會得到：
  https://law-format-converter.pages.dev
